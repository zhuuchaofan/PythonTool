# 🐍 Python 小工具集

> 一个用于存放日常 Python 实用脚本和小工具的仓库。

---

## 📁 目录结构

```
PythonTool/
├── README.md           # 项目说明
├── CONTRIBUTING.md     # 开发规范
├── requirements.txt    # 依赖管理
├── tools/              # 工具脚本目录
│   └── example/        # 示例工具
└── utils/              # 公共工具函数
```

---

## 🚀 快速开始

### 环境要求

- Python 3.10+
- pip

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行工具

```bash
python tools/<tool_name>/main.py
```
